A strongly typed functional programming language that compiles to JavaScript.

Browser

space-invaders-ps by @quephird

PureScript provides bindings to the web APIs available in the browser, making it an excellent choice for developing part or all of a traditional web application.

Server

Hyper by @owickstrom

PureScript can run server-side on NodeJS, or in any JavaScript environment which supports the CommonJS module standard. You can write both client and server in PureScript, and share code between the two.

Apps

React Native bindings by @doolse

PureScript works well with JavaScript-based tools such as React Native or Electron, which makes it possible to develop cross-platform applications using all of the expressive power that comes with using PureScript.

And More!

pure11 by @andyarvanitis

PureScript is not limited to just JavaScript! Alternative backends exist for C++11 and Erlang, and the PureScript compiler is built with extensibility in mind, so you can target your favorite language easily.

Hello, DOM!

Try this example now

PureScript's expressive type system and lightweight syntax make it simple to define domain-specific languages, which can be used to solve problems like templating the DOM. Bindings also exist for libraries such as React and Virtual DOM.

import Flare
import Flare.Smolder
 
ui = greet <$> string "Name" "World"
  where
  greet name = h1 $ text $ "Hello, " <> name <> "!"

Higher-order Functions

Try this example now

Higher-order functions allow the developer to write fluent, expressive code. Here, the higher-order function thrice is being used to simplify some code while working with the canvas.

import Flare
import Flare.Drawing

thrice f x = f (f (f x))

scene = thrice subdivide
      $ outlined (outlineColor black)
      $ frame 0.0 0.0 1.0

Testing

Try this example now

PureScript provides a form of ad-hoc polymorphism in the form of type classes, inspired by Haskell. Type classes are used in the QuickCheck and StrongCheck libraries to support generative testing, which separates test definitions from the generation of test cases.

import Test.QuickCheck

main = do
  quickCheck \xs ys ->
    isSorted $ merge (sort xs) (sort ys)
  quickCheck \xs ys ->
    xs `isSubarrayOf` merge xs ys
